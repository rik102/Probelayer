# Skills Context

## Installed Skills

- `gpt-taste`
- `high-end-visual-design`
- `full-output-enforcement`

## How To Apply

When building UI or revising product surfaces:

- Use `gpt-taste` to enforce layout variance, GSAP motion, AIDA structure, and anti-slop execution.
- Use `high-end-visual-design` to keep the UI premium, polished, and spacious.
- Use `full-output-enforcement` to avoid partial deliverables and placeholder code.

## Design Requirements

- `DESIGN_VARIANCE`: 9/10
- `MOTION_INTENSITY`: 9/10
- `VISUAL_DENSITY`: 4/10
