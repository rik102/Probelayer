# Probelayer Context

## Product

Probelayer is a synthetic human failure testing platform for pre-release UX, trust, accessibility, cognitive load, and defensive pen-testing.

## Design Dials

- `DESIGN_VARIANCE`: 9/10
- `MOTION_INTENSITY`: 9/10
- `VISUAL_DENSITY`: 4/10

## Mandatory Build Principles

- High layout variance with intentional asymmetry
- Strong motion choreography and meaningful reveals
- Premium, calm, expensive-feeling surfaces
- Low-mid information density per viewport
- No placeholder sections, stub copy, or half-finished UI
- Full implementation only

## Skills In Use

- `gpt-taste`
- `high-end-visual-design`
- `full-output-enforcement`

## Model Runtime

- `OPENAI_COMPAT_CHAT_URL` points to the Gemini OpenAI-compatible endpoint
- `OPENAI_COMPAT_API_KEY` contains the Gemini API key
- `VISION_MODEL=gemini-3.5-flash`

## Default Personas

The default persona set covers:

- Low-tech older adult
- Distracted single parent
- Impatient shopper
- Non-native English speaker
- Visually overwhelmed user
- Scammer / exploit seeker
- First-time mobile user
- Security-conscious enterprise buyer
- Neurodivergent / executive-function sensitive user
- Keyboard-only accessibility user
